The Pipeline

Six agents.
One sentence input.

Each agent is a specialist. Together they form an autonomous content factory that runs while you sleep.

01
💡

Ideas Discovery

Scours HackerNews, Google Trends, and the web to surface trending, high-novelty topics. Avoids repeats with a Firestore memory layer.

02
🔍

Deep Research

Multi-angle web search via Tavily, Firecrawl, and DuckDuckGo. Extracts key facts, statistics, and quotable moments into a structured brief.

03
📝

Script Generation

Writes hook-first, 60–75 second YouTube Shorts scripts optimized for your niche, tone, and creator profile. No filler, no fluff.

04
🎬

Video Production

TTS voiceover with Edge TTS or ElevenLabs. AI image generation via Gemini Imagen. Assembled with ffmpeg and uploaded directly to YouTube.

05
📅

Smart Scheduling

Finds the optimal publish time for your niche and audience. Creates a Google Calendar event and schedules the YouTube post automatically.

06
📊

Analytics Loop

48 hours after publish, fetches views, CTR, watch time, and engagement. Feeds results back to improve future topic scoring. It learns.

Capabilities

Built for serious
content creators.

Not a toy. Hermes is production infrastructure for creators who want to publish consistently without burning out.

🎙️

Natural Language Control

Describe what you want in plain English. The coordinator agent understands intent, niche, deadline, and style — no form-filling required.

🧠

Creator Profiles

Every creator has a unique voice. Configure your niche, tone, pacing, call-to-action style, and TTS voice once — applied to every video.

🔁

Self-Improving

The analytics agent feeds post-publish performance back into the topic scoring system. Topics that perform well get surfaced more often.

☁️

Fully Cloud-Native

Runs on Google Cloud Run with Firestore storage. Scales to zero when idle. Deploy once, run forever. Integrates with Cloud Scheduler for cron jobs.

💬

Interactive Chat Mode

Ask questions, request research, get topic ideas, or run one-off scripts via the chat interface. The coordinator holds full context across the session.

🔌

Open API

Every feature is exposed via clean REST endpoints. Build your own integrations, triggers, or workflows on top of the pipeline with full Swagger docs.
